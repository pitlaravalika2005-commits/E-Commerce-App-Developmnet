# E-Commerce-App-Developmnet

## 📌 Description
A simple E-Commerce website built using HTML, CSS, and JavaScript. It allows users to view products, add items to cart, and calculate total price.

---

## 🚀 Features
- Product listing
- Add to cart
- Total price calculation
- Checkout alert
- Simple and colorful UI

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript

---

## 📁 Files
- index.html
- style.css
- script.js

---

## ▶️ How to Run
1. Download the project folder
2. Open `index.html` in a browser
3. Start using the website

---

## 👨‍💻 Output
- Product cards displayed
- Cart updates dynamically
- Total price shown
- Checkout message displayed

---

## 📌 Note
This is a front-end only project (no database or backend).
